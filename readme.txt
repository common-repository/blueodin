Blue Odin Companion is a plugin to enable additional features in the Blue Odin service

== Description ==

Designed for eCommerce, BlueOdin automates your email marketing to help you build a strong relationship with your clients
and as a result increase orders, order value and customer satisfaction. How? Schedule automatic follow ups, seasonal
promotions, segment customers for highly relevant emails, send coupons to incentivize customers & more.

The Blue Odin Companion plugin helps provide more features for your Blue Odin dashboard

* Assign Google Analytics UTM values to orders, to see where your order traffic is coming from.
* Capture abandoned cart information, so you can save potentially lost sales.
